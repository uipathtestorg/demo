Release Notes

SAP v1.0.0 - Released on September 22, 2025

Executive Summary

This release introduces foundational utilities for managing order data in SAP, ensuring clear and auditable processes for creating and amending orders. These updates aim to enhance operational efficiency and data integrity.

New Features

Order Maintenance Utilities (Procedural ABAP)

This feature introduces procedural ABAP reports to create and amend orders in the custom DDIC table ZORDERS. Key highlights include:

  • Separate actions for creating and amending orders to prevent accidental overwrites.
  • Input validation for non-negative amounts.
  • Timestamping for creation and amendments.
  • Comprehensive ABAP Unit tests with over 90% coverage.

Create Order (Error if Exists)

Allows back-office users to create new orders in ZORDERS with accurate customer and amount details. Key acceptance criteria include:

  • Successful creation of new orders with proper validation.
  • Error handling for duplicate orders and negative amounts.
